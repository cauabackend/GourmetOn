# GourmetOn - Landing Page

## Descrição

Landing Page responsiva para app de delivery GourmetOn, com integração à Spoonacular API para exibir receitas em destaque.

## Tech Stack

- React 18
- Vite
- Tailwind CSS v4
- Spoonacular API
- Vercel (Deploy)

## Como Rodar Localmente

```bash
git clone <repo>
cd gourmet-on
npm install
npm run dev
```

Crie um arquivo `.env.local` com:

```
VITE_API_KEY=sua_chave_spoonacular_aqui
```

## Features

- Hero Section com CTA
- Menu fixo com scroll suave
- Seção de Benefícios com cards
- Galeria de receitas da Spoonacular API
- Depoimentos de clientes
- Formulário de contato
- Footer com links
- Responsivo (mobile/tablet/desktop)
- Deploy na Vercel
